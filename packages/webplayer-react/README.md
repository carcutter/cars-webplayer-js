# React WebPlayer

Use CarCutter WebPlayer in your React app

## Usage

1. Install package: `npm install @car-cutter/react-webplayer`
2. Import: `import { WebPlayer } from "@car-cutter/react-webplayer"`
3. Use: `<WebPlayer compositionUrl={url} />`
