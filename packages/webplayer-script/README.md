# Script WebPlayer

Use CarCutter WebPlayer without any Package Manager

## Usage

1. Import script in `<head>` : `<script src="https://cdn.car-cutter.com/libs/web-player/v3/bundle.js"></script>`
2. Use: `<cc-webplayer composition-url="..." />`
